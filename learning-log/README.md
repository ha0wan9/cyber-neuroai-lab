# Learning Log

This folder contains chronological learning records.

A learning log is not a polished note. It captures the evolution of understanding during a specific session, day, or week.

## Purpose

Learning logs are used to:

- preserve raw learning progress;
- record confusion before it disappears;
- track conceptual evolution;
- identify recurring questions;
- generate structured notes later;
- support weekly and monthly reviews.

## Recommended Format

Each learning log should include:

- date;
- material studied;
- core question;
- key concepts;
- current understanding;
- unclear points;
- links to AI, neuroscience, engineering, or philosophy;
- possible outputs.

## Lifecycle

```text
Learning log
→ structured note
→ concept note
→ experiment or research question
→ weekly review
```

## Rule

Do not over-polish learning logs.

Their value is to preserve the cognitive trace of learning.
