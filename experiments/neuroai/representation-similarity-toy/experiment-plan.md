# Experiment Plan

## Aim

Create a small synthetic RSA workflow that compares representational geometries across toy systems.

## Steps

1. Generate a set of synthetic stimuli or stimulus labels.
2. Create two representation matrices from a shared latent structure.
3. Create a third unrelated representation as a baseline.
4. Compute pairwise similarity matrices.
5. Compare similarity matrices with a simple summary statistic.
6. Save compact summaries under `outputs/`.

## Constraints

- Do not use external model downloads.
- Avoid large notebooks or binary artifacts.
- Keep the metric definitions explicit.

## Verification

Run the command documented in `README.md` after the script exists.
