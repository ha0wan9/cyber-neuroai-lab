# Deep Learning Notes

This folder is a bridge layer between deep learning engineering knowledge and NeuroAI questions.

It is not intended to become a generic deep learning course. The purpose is to keep only the concepts that help interpret, train, compare, or critique artificial neural networks as possible models of neural and cognitive systems.

## Purpose

- Translate familiar AI engineering concepts into brain-modeling questions.
- Identify where deep learning analogies are useful, weak, or misleading.
- Connect computational neuroscience concepts to artificial neural network practice.
- Produce small experiments that support NeuroAI learning rather than benchmark chasing.

## Core Questions

- When can an artificial neural network be treated as a model of a biological system?
- Which training objectives, architectures, and representations matter for brain-model comparison?
- What does optimization explain, and what does it obscure?
- How should engineering success be separated from biological plausibility?
- What minimal experiments can test the limits of a brain-AI analogy?

## Note Template

Each note should answer:

1. Why this matters for NeuroAI.
2. What an AI engineer may already know.
3. What must be reinterpreted through a brain-modeling lens.
4. Connection to computational neuroscience.
5. Failure mode: how this concept can mislead brain-AI comparison.
6. Minimal experiment idea.
