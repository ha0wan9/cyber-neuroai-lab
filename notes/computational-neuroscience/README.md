# Computational Neuroscience Notes

This folder contains structured notes on computational neuroscience.

## Purpose

The goal is to build a usable conceptual and mathematical foundation for NeuroAI and computational cognitive neuroscience.

This is not meant to be a complete textbook. It is a structure-first learning map.

## Core Questions

- How can neural systems be modeled computationally?
- How do neurons encode information?
- How do populations of neurons represent variables?
- How do neural dynamics evolve over time?
- How can we decode stimulus, action, or cognition from neural data?
- How do learning and plasticity change neural computation?
- How can computational models connect neural activity to behavior?

## Learning Rule

Each note should answer:

1. What problem does this concept solve?
2. What are the key variables?
3. What is the mechanism?
4. What is the mathematical form?
5. What is a failure mode or common misconception?
6. How does it connect to AI or NeuroAI?
