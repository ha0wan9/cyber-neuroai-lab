# Cognitive Systems Notes

This folder contains notes on cognition, external memory, human-AI interaction, and agent-assisted research systems.

## Purpose

The goal is to study how cognitive processes can be extended, structured, and augmented through tools, workflows, and AI agents.

This is the system-level layer of Cyber NeuroAI Lab.

## Core Questions

- How do humans manage attention, memory, and reasoning under limited cognitive bandwidth?
- How can external tools serve as cognitive scaffolds?
- How should AI systems collaborate with human reasoning?
- What makes a human-AI workflow reliable rather than merely fluent?
- How can learning systems preserve insight without overloading working memory?
- How can personal knowledge systems become research infrastructure?

A good cognitive system does not simply store information.

It helps transform:

```text
fragmented input
→ structured understanding
→ testable questions
→ durable artifacts
→ future research directions
```
