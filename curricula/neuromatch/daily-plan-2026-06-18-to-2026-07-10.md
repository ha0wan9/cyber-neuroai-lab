# Pre-NeuroAI Daily Learning Plan

Date range: 2026-06-18 to 2026-07-10

Purpose: compress the pre-NeuroAI period into 5 cycles of structured learning. Each cycle has 3 learning days, each learning day is 3 hours, and each day must produce a durable repository artifact.

Learner profile: AI/CV engineer transitioning toward NeuroAI and computational cognitive neuroscience.

Style constraint: structure-first. Every day ends with a small written artifact, map, stub, or review note that can be reused later.

Primary sources:

- Neuromatch Computational Neuroscience Coursebook: <https://compneuro.neuromatch.io/>
- Neuromatch Computational Neuroscience Schedule: <https://compneuro.neuromatch.io/tutorials/Schedule/daily_schedules.html>
- Neuromatch Deep Learning Coursebook: <https://deeplearning.neuromatch.io/tutorials/intro.html>
- Neuromatch NeuroAI Prerequisites: <https://neuroai.neuromatch.io/prereqs/NeuroAI.html>

## Cycle Overview

| Cycle | Dates | Neuromatch focus | Deep learning bridge | Cycle output |
|---|---:|---|---|---|
| 1 | 2026-06-18 to 2026-06-22 | W0D0 Neuro Video Series; W1D1 Model Types | AI/CV engineer orientation to neuroscience data and model purposes | Measurement modality map + What/How/Why model note |
