# Neuromatch Learning Path

This directory tracks official Neuromatch sources that support the repository's learning path from computational neuroscience foundations toward NeuroAI.

## Purpose

The directory is a source-control layer for curriculum inputs, not a course summary. It records which Neuromatch materials should feed into notes, experiments, papers, learning logs, and future research-planning artifacts.

## Registry

- [`source-registry.md`](source-registry.md) is the canonical registry for official Neuromatch coursebook and prerequisite sources used in this learning path.
- [`content-index/`](content-index/) is a structured per-tutorial index of all learning content across the three coursebooks (title, objectives, sections, source URL). It is a catalog, not a content copy.

## Use Guidelines

When working from a registered source:

1. Treat the source as curriculum input, not as a permanent note by itself.
2. Convert completed learning work into durable artifacts under the relevant repository folders.
3. Keep claims traceable to the official source URL or to primary literature when a scientific claim becomes important.
4. Separate established facts, interpretation, working hypotheses, analogies, speculation, and open questions in downstream notes.
