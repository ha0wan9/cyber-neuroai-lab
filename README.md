# Cyber NeuroAI Lab

Cyber NeuroAI Lab is a personal research and learning repository for NeuroAI, computational neuroscience, cognitive systems, and AI-assisted scientific workflows.

This repository is not only a note archive. It is intended to become a long-term external cognition system: a place where learning, experiments, paper reading, research questions, and public-facing ideas are progressively compressed into reusable knowledge assets.

## Core Idea

The central thesis of this repository is that modern AI can accelerate scientific learning and research, but only if human reasoning remains responsible for:

- framing the right questions;
- checking assumptions;
- building conceptual structure;
- validating claims against evidence;
- turning scattered insights into durable knowledge artifacts.

This repository is therefore designed as a version-controlled learning and research lab.

## Main Directions

### 1. Computational Neuroscience

Building the mathematical and conceptual foundations needed to understand neural computation.

Focus areas:

- neural coding;
- dynamical systems;
- probability and statistics for neural data;
- encoding and decoding models;
- dimensionality reduction;
- decision-making models;
- learning and plasticity;
- recurrent neural networks.

### 2. NeuroAI

Studying the interaction between artificial neural networks and biological neural systems.

Focus areas:

- brain-model alignment;
- neural representation analysis;
- encoding and decoding models;
- artificial neural networks as brain models;
- brain-guided AI model design;
- multimodal representation alignment;
- cognitive and behavioral modeling.

### 3. Cognitive Systems and Human-AI Interaction

Exploring how human cognition, external memory, agent workflows, and AI systems can be combined into more powerful research and learning loops.

Focus areas:

- external cognition;
- AI-assisted learning;
- human-AI co-reasoning;
- agentic research workflows;
- memory and attention augmentation;
- cognitive interfaces for scientific work.

## Repository Structure

```text
learning-log/     Chronological learning records and weekly reviews
notes/            Structured notes by domain
experiments/      Code, notebooks, simulations, and small research prototypes
papers/           Paper summaries, reading notes, and literature maps
agents/           AI-assisted research workflows and prompts
templates/        Reusable templates for notes, experiments, and reviews
assets/           Figures, diagrams, screenshots, and other static resources
```

## Current Phase

The current phase focuses on prerequisite learning for NeuroAI, starting with computational neuroscience.

Initial objectives:

- build mathematical intuition for neural computation;
- complete selected computational neuroscience prerequisite materials;
- transform course material into reusable notes and experiments;
- create a research question backlog;
- develop a personal research direction at the intersection of NeuroAI, cognitive science, and AI-assisted research.

## Working Principle

Every learning unit should ideally produce at least one durable artifact:

- a short conceptual note;
- a runnable code experiment;
- a paper summary;
- a question map;
- a research hypothesis;
- a learning log entry;
- a future blog or essay draft.

The goal is not to collect information, but to progressively build a research-grade cognitive scaffold.

## Learning Loop

The default loop is:

```text
Raw learning input
→ working memory
→ structured note
→ concept extraction
→ experiment or question
→ weekly review
→ long-term memory commit
```

## AI-Assisted Workflow

AI tools may be used to:

- decompose learning materials;
- generate first-pass explanations;
- propose learning routes;
- identify missing assumptions;
- create flashcards or review questions;
- draft summaries;
- suggest experiments;
- generate pull requests for repository updates.

However, AI-generated content should be treated as a working draft. Important claims should be checked, revised, and connected to primary sources whenever possible.

## Long-Term Goal

The long-term goal is to turn this repository into a public intellectual and technical profile around:

- NeuroAI;
- computational cognitive neuroscience;
- brain-AI alignment;
- human-AI interfaces;
- AI-assisted research systems;
- independent research and development.
